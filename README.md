A simple Express + EJS web app that displays philosophers, philosophies, and quotes using shared partials, custom error pages, and a clean Bootstrap-style layout. The project is fully deployed on Render and demonstrates routing, views, navigation, and production-ready configuration.This app was created with the intent of making a more fun/engaging environment to learn about the chosen topic for those who are interested.

How to run locally:
npm install
npm start

Then open:
http://localhost:3000 (VSCode)
